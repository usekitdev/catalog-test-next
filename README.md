
Задача: Создать страницу каталога товаров на Next.js с использованием React-компонентов.
Требуемая реализация:
Страница с сеткой товаров (данные можно моковые или из публичного API).
Реализация функционала Infinite scroll (подгрузка товаров при скролле).
Разработка компонента карточки товара с использованием стилей и компонентов shadcn/ui.
Документирование основных компонентов в Storybook.
Использование TypeScript с типизацией данных.
Приложение файла README с инструкцией по запуску проекта.

# Catalog App

## Запуск проекта

### Установка зависимостей
npm install

### Запуск dev сервера
npm run dev

### Запуск Storybook
npm run storybook
